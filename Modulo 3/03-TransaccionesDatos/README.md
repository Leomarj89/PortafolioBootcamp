# Punto 3 - Manipulación de Datos (DML)

Este archivo contiene instrucciones SQL para modificar los datos en las tablas ya creadas, demostrando el uso de sentencias `UPDATE` y `DELETE` como parte del lenguaje de manipulación de datos (DML).

## Operaciones realizadas

1. **UPDATE**
   - Cambiar el correo electrónico de un cliente.
   - Modificar el precio de un producto.
   - Ajustar la cantidad de un producto en un pedido.

2. **DELETE**
   - Eliminar un pedido cancelado (junto a sus detalles).
   - Eliminar un cliente sin pedidos.

## Ejecución

- Ejecutar los scripts después de haber insertado los datos (`insertar_datos.sql`).
- Asegúrate de respetar el orden (por ejemplo, eliminar detalles de pedidos antes que el pedido).
- Puedes personalizar los valores según tus pruebas.

Estas operaciones completan la manipulación básica de registros dentro de una base de datos relacional.
