# Punto 4 - DDL para mantención y modificación de estructuras

Este archivo contiene ejemplos de uso del Lenguaje de Definición de Datos (DDL) para modificar y mantener objetos dentro de una base de datos relacional.

## Operaciones realizadas

- Agregar columnas a una tabla existente (`ALTER TABLE`).
- Modificar tipos de datos de columnas.
- Crear y eliminar índices (`CREATE INDEX`, `DROP INDEX`).
- Eliminar una tabla con seguridad (`DROP TABLE IF EXISTS`).

Estas operaciones son clave para mantener una base de datos actualizada según las necesidades del negocio u organización.

## Ejecución

Ejecutar el archivo `modificar_estructuras.sql` una vez que las tablas principales estén creadas.
