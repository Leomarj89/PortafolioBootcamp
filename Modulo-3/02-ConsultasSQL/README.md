# Punto 2 - Consultas SQL

Este archivo contiene varias consultas SQL que demuestran el uso del Lenguaje Estructurado de Consultas (SQL) para obtener información relevante desde una base de datos relacional.

## Consultas incluidas

1. Pedidos realizados por un cliente específico.
2. Productos comprados por un cliente con cantidad.
3. Total gastado por cada cliente.
4. Cantidad de pedidos por día.
5. Productos más vendidos.
6. Pedidos que contienen más de un producto.

## Ejecución

1. Asegúrate de tener creadas y pobladas las tablas de la base de datos `tienda`.
2. Ejecuta `consultas_select_join.sql` en tu cliente SQL.
3. Puedes modificar los nombres de clientes o productos en los filtros `WHERE` para probar otras combinaciones.

Estas consultas utilizan las cláusulas:
- `SELECT`
- `JOIN`
- `WHERE`
- `GROUP BY`
- `ORDER BY`
- `HAVING`

Y muestran cómo extraer valor desde datos organizados relacionalmente.
